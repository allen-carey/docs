## Introduction
    
You have the option to export your Web Tests.
    
<Info> The option to export Web Tests in executable format has been removed. </Info>
    
## 1. JSON File
    
This option allows you to export your test as a JSON file, which can be used to import it into other systems or to store it in a repository.
    
The JSON file contains all the data from each step.
    
<Info> Tests exported in JSON format cannot be executed. </Info>
