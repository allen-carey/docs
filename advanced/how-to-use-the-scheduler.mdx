## Introduction

You can easily schedule test executions by using our <a class="endtest_link" href="https://app.endtest.io/scheduler">Scheduler</a>.

## Steps
1. Click on the `Add to Scheduler` button from the `Run Test Suite` modal.
2. Add a name for this job in the `Schedule Name` input.
3. Select the days in which you would like your test to be executed.
4. Select the `Schedule Type`.
5. Select the time or the interval at which you would like your test to be executed.
6. Click on the `Save Scheduled Test Execution` button.

You can see the entire list of scheduled test executions in the <a class="endtest_link" href="https://app.endtest.io/scheduler">Scheduler</a> section.
